## 🇧🇷 Projetos que estou desenvolvendo atualmente:

### 1. Audivise: Assistente de Enquadramento para Videoconferências
**Descrição:** Desenvolvido de forma independente para o Projeto Integrador V da graduação em Engenharia de Computação da Univesp, este é um sistema assistivo baseado em Internet das Coisas (IoT) e Visão Computacional, projetado para fornecer autonomia a pessoas com deficiência visual durante videoconferências. O sistema detecta a posição do rosto do usuário e fornece feedback auditivo em tempo real para auxiliar no enquadramento e centralização diante da câmera. O processamento de imagem ocorre localmente no navegador (Edge Computing) para garantir total privacidade, e a lógica de enquadramento foi verificada matematicamente usando métodos formais.
**Tecnologias Utilizadas:** Rust (Serverless Functions), HTML5, CSS3, JavaScript, MediaPipe Face Landmarker, Lean 4 (Verificação Formal), Web Speech API, Vercel.
**Repositório:** https://github.com/eliezer-tavares/audivise

### 2. PrevisIA: Previsor e Guia de Acessibilidade Web
**Descrição:** Este projeto é uma ferramenta que prevê escores de acessibilidade web usando machine learning (Redes Neurais com PyTorch) e gera guias narrativos em linguagem natural para deficientes visuais que utilizam leitores de tela, como NVDA. Desenvolvida para a disciplina Projeto Integrador IV da graduação em Engenharia de Computação da Univesp, combina análise estática e dinâmica de páginas web.
**Tecnologias Utilizadas:** Python, Flask, PyTorch, BeautifulSoup, Playwright, Axe, SpeechSynthesis, Babel, Tranco.
**Repositório:** https://github.com/eliezer-tavares/previsia-acessibilidade

### 3. AudioLangAnalyzerApp
**Descrição:** O AudioLangAnalyzerApp é uma aplicação desktop desenvolvida em Python com wxPython, projetada para analisar arquivos de áudio (MP3) com foco total em acessibilidade. A ferramenta valida o áudio, transcreve a fala usando o modelo Whisper, detecta o idioma, estima a fluência do falante, traduz para o Português e gera um relatório completo em PDF.
**Tecnologias Utilizadas:** Python, wxPython, pydub (FFmpeg), SpeechRecognition (OpenAI Whisper), langdetect, deep-translator, reportlab, pytest.
**Artigo sobre o projeto no Linkedin:** https://www.linkedin.com/pulse/desenvolvendo-com-prop%C3%B3sito-como-criei-um-analisador-eliezer-auwcf/
**Repositório:** https://github.com/eliezer-tavares/AudioLangAnalyzerApp

### 4. Monitor de Disponibilidade de Sites
**Descrição:** Este projeto é um monitor de disponibilidade de sites que verifica periodicamente o status de uma lista de sites e envia alertas por e-mail e Telegram (opcional) caso algum site fique offline. Possui API REST, interface web para visualização em tempo real, armazenamento de histórico em SQLite e é dockerizado para fácil implantação.
**Tecnologias Utilizadas:** Python, Flask, FastAPI, Uvicorn, Asyncio, Requests, Aiosmtplib, Docker, SQLite.
**Repositório:** https://github.com/eliezer-tavares/Monitoramento-de-Sites

### 5. Scanner de Vulnerabilidades em Redes
**Descrição:** Script Python para escanear vulnerabilidades em uma rede, identificando dispositivos conectados via ARP, escaneando portas abertas com Nmap e consultando a API do National Vulnerability Database (NVD) para verificar vulnerabilidades conhecidas associadas aos serviços detectados. Inclui suporte a saída de áudio para acessibilidade.
**Tecnologias Utilizadas:** Python, Scapy, python-nmap, Requests, pyttsx3, NVD API.
**Repositório:** https://github.com/eliezer-tavares/Network-Vulnerability-Scanner

## 🇺🇸 Projects I am currently developing:

### 1. Audivise: Assistive Framing System for Videoconferences
**Description:** Developed independently for the Integrative Project V of the Computer Engineering degree at Univesp, this is an assistive system based on the Internet of Things (IoT) and Computer Vision, designed to provide autonomy to visually impaired people during video conferences. The system detects the user's face position and provides real-time audio feedback to assist with framing and centering in front of the camera. Image processing occurs locally in the browser (Edge Computing) to ensure total privacy, and the framing logic was mathematically verified using formal methods.
**Technologies Used:** Rust (Serverless Functions), HTML5, CSS3, JavaScript, MediaPipe Face Landmarker, Lean 4 (Formal Verification), Web Speech API, Vercel.
**Repository:** https://github.com/eliezer-tavares/audivise

### 2. PrevisIA: Web Accessibility Predictor and Guide
**Description:** This tool predicts web accessibility scores using machine learning (Neural Networks with PyTorch) and generates narrative guides in natural language for visually impaired users who use screen readers, such as NVDA. Developed for the Integrative Project IV course in the Computer Engineering degree at Univesp, it combines static and dynamic analysis of web pages.
**Technologies Used:** Python, Flask, PyTorch, BeautifulSoup, Playwright, Axe, SpeechSynthesis, Babel, Tranco.
**Repository:** https://github.com/eliezer-tavares/previsia-acessibilidade

### 3. AudioLangAnalyzerApp
**Description:** AudioLangAnalyzerApp is a desktop application developed in Python with wxPython, designed to analyze audio files (MP3) with a total focus on accessibility. The tool validates the audio, transcribes speech using the Whisper model, detects the language, estimates the speaker's fluency, translates it to Portuguese, and generates a complete PDF report.
**Technologies Used:** Python, wxPython, pydub (FFmpeg), SpeechRecognition (OpenAI Whisper), langdetect, deep-translator, reportlab, pytest.
**Article about the project on LinkedIn:** https://www.linkedin.com/pulse/desenvolvendo-com-prop%C3%B3sito-como-criei-um-analisador-eliezer-auwcf/
**Repository:** https://github.com/eliezer-tavares/AudioLangAnalyzerApp

### 4. Website Availability Monitor
**Description:** A website availability monitor that periodically checks the status of a list of sites and sends alerts via email and Telegram (optional) if any site goes offline. It features a REST API, a web interface for real-time visualization, history storage in SQLite, and is dockerized for easy deployment.
**Technologies Used:** Python, Flask, FastAPI, Uvicorn, Asyncio, Requests, Aiosmtplib, Docker, SQLite.
**Repository:** https://github.com/eliezer-tavares/Monitoramento-de-Sites

### 5. Network Vulnerability Scanner
**Description:** Python script to scan vulnerabilities in a network, identifying connected devices via ARP, scanning open ports with Nmap, and querying the National Vulnerability Database (NVD) API to check for known vulnerabilities associated with the detected services. Includes audio output support for accessibility.
**Technologies Used:** Python, Scapy, python-nmap, Requests, pyttsx3, NVD API.
**Repository:** https://github.com/eliezer-tavares/Network-Vulnerability-Scanner
